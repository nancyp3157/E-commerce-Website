# E-commerce-Website
This is a basic e-commerce website project.

It includes user login, signup, wishlist, add to cart, and product view using a modal.

A simple backend is used to store user data so that every time a user logs in, their previous data is loaded.

<h3>Features:</h3>


User Signup and Login

Backend storage for user data

Cart data saved for each user

Wishlist data saved for each user

Add to Cart functionality

Product details shown in a modal popup

User data is restored on every login


<h3>How It Works:</h3>


User creates an account using signup

User logs in using saved credentials

Backend returns stored user data

Cart and wishlist are restored

User can add or remove products

All changes are saved automatically


<h3>Frontend:</h3>


HTML

CSS

JavaScript



This project is created for learning purposes and uses a simple backend setup.
